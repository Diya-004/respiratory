About PneumoScan

An AI-powered respiratory screening interface rebuilt on top of the new recording-level backend. The deployed model is the pneumonia-focused strong CNN, evaluated with multi-window aggregation across the uploaded audio.

Platform Overview

What PneumoScan does

01
🎙️
Audio Ingestion
Accepts lung sound recordings in WAV, MP3, FLAC, OGG or M4A format. The backend trims, resamples, and evaluates the recording without needing manual clip selection.
02
🧠
Recording-Level Inference
A custom strong CNN analyses multiple 5-second windows from the same recording, then averages probabilities to produce a more stable diagnosis than a single-clip prediction.
03
📊
Transparent Outputs
Returns class probabilities, severity, representative window details, and a real Grad-CAM attention heatmap generated from the uploaded recording.
Detected Conditions

Four respiratory classes

●

Normal

No abnormal wheeze or crackle pattern is dominant in the analysed recording. This is the healthy baseline class in the current model.

●

Asthma

Wheeze-dominant respiratory sounds associated with airway narrowing and bronchospasm. The model treats this as an abnormal but potentially reversible obstructive pattern.

●

Pneumonia

Crackle-dominant patterns that may indicate infection, fluid, or consolidation. This remains the hardest class and was the focus of the final retraining run.

●

COPD

Obstructive lung disease patterns with strong chronic airflow limitation cues. This is currently the most confidently learned class in the deployed model.
